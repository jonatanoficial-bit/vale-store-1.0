Como recebo o app depois de pagar?

Você paga no provedor (Ton/PagBank/etc.) e clica em “Já paguei” para gerar um código de compra. Envie o código + comprovante para o vendedor. Depois ele libera um código de entrega para você usar na tela de entrega.

Como instalar APK no Android?

Veja o guia passo a passo: Como instalar APK.

Posso rebaixar/compartilhar o link do APK?

Não recomendamos. O fluxo de entrega é por código, e o vendedor pode trocar links/versões a qualquer momento.

Sem investimento: fluxo MVP

Este checkout foi desenhado para funcionar sem backend e sem custo. Você coloca um link de pagamento (Ton/PagBank/Asaas etc.) no produto e, após pagar, o comprador recebe um código de compra para apresentar ao vendedor. O vendedor confirma e libera o código de entrega.
