Política de Privacidade

Versão MVP (0 investimento) — sem backend por padrão.

1. Como seus dados são usados

Neste MVP, o site pode armazenar informações localmente no seu navegador via localStorage para melhorar a experiência (ex.: biblioteca, pedidos gerados, entregas liberadas no admin).

2. O que NÃO coletamos

  • Não coletamos dados pessoais em servidor (por padrão).
  • Não usamos cookies obrigatórios no core do produto.
  • Não rastreamos navegação com analytics pago.

3. Pagamentos

Pagamentos são realizados em páginas externas (links de pagamento). O Vale Games Store não processa nem armazena dados do cartão.

4. Alterações

Esta política pode evoluir quando um backend for integrado (ex.: para confirmar pagamentos automaticamente).
